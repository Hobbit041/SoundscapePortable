# Soundscape — Standalone App

Портированный микшер из Foundry VTT в виде самостоятельного приложения на Electron.

---

## Быстрый старт

### 1. Установи Node.js
Скачай с https://nodejs.org (версия 18 или новее)

### 2. Установи зависимости
Открой папку проекта в терминале и выполни:
```
npm install
```

### 3. Запусти приложение
```
npm start
```

---

## Сборка в .exe (Windows)
```
npm run build
```
Готовый установщик появится в папке `dist/`.

---

## Как пользоваться

### Каналы микшера
- **Папка** (иконка) — выбрать аудиофайл для канала
- **Drag & drop** — перетащи файл из проводника прямо на канал
- **M** — Mute (заглушить)
- **S** — Solo (только этот канал)
- **L** — Link (связать громкость с соседними каналами)
- **Фейдер** — громкость канала (0–125%)
- **Pan** — стерео панорама
- **▶/■** — воспроизведение/стоп отдельного канала

### Саундпад
- **Левый клик** — воспроизвести звук
- **Правый клик** — назначить аудиофайл
- **Drag & drop** — перетащить файл прямо на кнопку
- Жёлтая рамка = звук в режиме loop

### Навигация по саундскейпам
- **← →** — переключать пресеты
- **+** — добавить новый
- **🗑** — удалить текущий
- **Имя** — редактируется прямо в поле

### Импорт / Экспорт
- **Экспорт**: сохраняет все саундскейпы в файл `.soundscapeData`
- **Импорт**: загружает файл `.soundscapeData` (совместим с оригинальным Foundry-модулем)

### Отсутствующие файлы
Если файл был перемещён или удалён, приложение спросит:
> "File not found: C:\... Would you like to locate it?"
Нажми OK и укажи новое расположение файла.

## MIDI

Приложение автоматически определяет подключённые MIDI-устройства.
Статус отображается в верхней панели.

### Маппинг по умолчанию
| MIDI CC | Действие |
|---------|----------|
| CC 0–7  | Громкость каналов 1–8 |
| CC 8    | Master volume |
| CC 9    | Soundboard volume |
| CC 64   | Play/Stop (≥64 = play) |
| Note 36–60 | Кнопки саундпада 0–24 |

Маппинг можно изменить в `renderer/src/midi.js` в методе `_defaultMapping()`.
