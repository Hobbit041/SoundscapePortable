# Dungeonscape — Soundscape Portable 

Портированный микшер из Foundry VTT в виде самостоятельного приложения на Electron. Ссылка на оригинальный модуль для FVTT: https://github.com/MaterialFoundry/Soundscape
Приложение для мастеров настольных ролевых игр, позволяющее удобно управлять звуками и музыкой на сессиях.
Поддерживает MIDI-устройства и управление с других устройств для более удобного управления. 

---

## Главные особенности
- Плавное и удобное управление всеми звуковыми файлами: музыкой, амбиентами, фоновыми и эпизодическими звуками
- Редактируемая очередь воспроизведения
- Звуковая панель, воспроизводящая привязанные к кнопке звуки в случайном (или поочередном) порядке
- Настраиваемые паузы между сменяемыми музыкальными треками
- Поддержка MIDI-контроллеров
- Возможность открыть доступ к приложению через браузер для управления через телефон или планшет.

## Как пользоваться

### Общее
- ** Click & hold** — перетаскивание дорожек

### Дорожки музыки
Предназначены для воспроизведения сменяемых звуковых файлов, в первую очередь музыки.
- **Папка** (иконка) — настройка дорожки
- **Drag & drop** — можно перетащить файл из проводника прямо на канал. По умолчанию перетаскиваемый файл(ы) заменяют очередь воспроизведения. Поведение меняется в настройках
- **M** — Mute (заглушить)
- **S** — Solo (только этот канал)
- **L** — Link (связать громкость с соседними каналами. Кликните на несколько дорожек для управления через один фейдер)
- **Фейдер** — громкость канала (0–125%)
- **▶/■** — воспроизведение/стоп отдельного канала
- ** < > ** — переключение играющих треков 

### Саундпад
- **Левый клик** — воспроизвести звук
- **Правый клик** — открыть настройки 
- **Drag & drop** — перетащить файл прямо на кнопку. Звуковые файлы добавляются в очередь воспроизведения, графические изображения устанавливаются иконкой кнопки

### Дорожки фоновых звуков
Файлы, добавленные на эти дорожки, всегда воспроизводятся зацикленно: звуки костра, дождя, воды и т.п.

### Навигация по профилям
- **☰** — список текущих пресетов
- **+** — добавить новый
- **🗑** — удалить текущий
- **Имя** — редактируется прямо в поле

### Сцены
Вы можете создавать сцены для аудиофайлов, в зависимости от потребностей: например, "Город", "Пещера", "Лес" и т.п. Для звуковой панели используются отдельные сцены, чтобы вы могли разделять звуки, например, "заклинания", "оружие" и т.п. 
Создав сцену, вы можете переименовать её, кликнув ПКМ по кнопке сцены. 

### Импорт / Экспорт
- **Экспорт**: сохраняет текущий профиль в файл `.soundscapeData`
- **Импорт**: загружает файл `.soundscapeData` (совместимость с оригинальным Foundry-модулем предполагается, но не гарантируется)

## MIDI

Приложение автоматически определяет подключённые MIDI-устройства.
Статус отображается в верхней панели.

### Маппинг 
Нажмите на панель MIDI, чтобы активировать режим привязки. Если ваши устройства определяются программой, то она способна создавать связь с переданными командами. Нажмите кнопку "скрепки", чтобы начать привязку. Зелёный цвет кнопки означает созданную привязку. Для фейдеров автоматически устанавливается связь с 0-127, для "крутилок" - в зависимости от типа, принимаются как CC 1/65, так и значения 1-127 - диспетчер определяет тип ввода автоматически.
